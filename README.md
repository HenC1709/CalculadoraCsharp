¡Bienvenido a mi primer proyecto pro en C#! Esta es una calculadora de consola diseñada para ser indestructible.

## ✨ Características
* **Operaciones básicas:** Suma, Resta, Multiplicación y División.
* **Validación de Datos:** Uso de `TryParse` para evitar que el programa explote con letras.
* **Seguridad Matemática:** Validación para evitar la división por cero.
* **Bucle Continuo:** No se cierra sola; tú decides cuándo terminar.
* **Personalización:** ¡Muchos colores y mensajes divertidos!

## 🚀 Tecnologías usadas
* Lenguaje: **C#**
* Framework: **.NET 10.0**
* Hardware de desarrollo: **HP Victus 15**

## 💻 Cómo ejecutarlo
1. Clona el repositorio.
2. Abre la terminal en la carpeta del proyecto.
3. Ejecuta el comando `dotnet run`.
